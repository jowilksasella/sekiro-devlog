Project Overview

This project implements a "Sekiro-like" combat system in Unreal Engine 5. Unlike traditional health-based combat, our core mechanics focus on the delicate dance of swords.

Core Architecture

2.1 Characters

  • ASekiroCharacter: The base class for all entities.
  • Inheritance: ACharacter -> ASekiroCharacter
  • Responsibilities: Input, Movement, Animation Playback.

2.2 Component System

The soul of the system is modular. Capabilities are granted via components:

P

USekiroPostureComponent

Manages the Posture bar and "Broken" states.

D

USekiroDeflectComponent

The shield against death. Calculates Parry vs. Block timing windows.

C

USekiroCombatComponent

Offensive capabilities. Hit detection (Line Trace) and Execution requests.

Key Mechanics Implementation

3.1 Posture & Stun

Attacks deal "Posture Damage". When Posture reaches max, the character enters a "Stunned" state.

Implementation: `SekiroPostureComponent` tracks the value. When broken, it broadcasts `OnPostureBroken`.

3.2 The Deflection Window

Precision is key. The `SekiroDeflectComponent` measures the time since `StartBlocking` was triggered.

  • < 0.2s: Perfect Parry (Sparks, High Posture Dmg)
  • > 0.2s: Block (Safe, but accumulate posture)
  • Not Blocking: Hit (Health Damage)

3.3 Deathblow (處決)

When an enemy is Stunned, the player can perform a "Deathblow". The SekiroCombatComponent detects the Stunned state and triggers the fatal ExecutionMontage on the attacker and DeathMontage on the victim.
